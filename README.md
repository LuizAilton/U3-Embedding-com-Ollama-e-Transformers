# U3-Embedding-com-Ollama-e-Transformers
Avaliação Online 3-Tokenização e Embeddings com Ollama e Transformers

## Sobre o Projeto
Este projeto demonstra, de forma prática, como realizar:
# Tokenização de texto utilizando modelos pré-treinados da biblioteca Transformers
# Geração de embeddings vetoriais com o modelo nomic-embed-text-v2-moe via Ollama
# Organização dos dados com Pandas

O objetivo é explorar como texto em linguagem natural pode ser transformado em representações numéricas úteis para tarefas de NLP.

## Conceitos Envolvidos
•	Tokenização
•	Embeddings (Representação vetorial)
•	Modelos pré-treinados
•	Processamento de linguagem natural

## Pré-requisitos
Antes de começar, você precisa ter instalado:
•	Python 3.8+
•	Pip
•	Ollama instalado na máquina

## Configuração do Ambiente

1. Instalar dependências
•	pip install ollama
•	pip install transformers
•	pip install pandas

2. Baixar modelo de embeddings no Ollama
ollama pull nomic-embed-text-v2-moe
